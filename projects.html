A mental health and self-improvement app built for the web. Using best practices in healthy communication, Hivemind users can add people from their IRL support network, build a well-worded and short request for help, and send their text in-app.

WeeMe


An app for everyone. Web application built for mobile first that allows users to input their current location and find nearby public / free-access bathrooms. The user can see each each bathroom's rating, and true or false values regarding each bathroom's: family friendliness, gender neutrality, and accessibility.

Technologies: Ruby on Rails, Boostrap, SASS, Devise, Rails Admin, Geocoder, GMaps4Rails, Rspec
